# Marketplace Copy - Piwigo Media

## App name

Piwigo Media

## Short description

Connect your Piwigo photo library to Canva.

## Long description

Piwigo Media lets you connect your own Piwigo gallery to Canva with the Piwigo Canva Connector plugin.

Browse albums, add photos to your Canva designs, and export finished designs back to a selected Piwigo album. Piwigo access is controlled by a connector token generated and revoked on your own Piwigo instance.

## Key benefits

- Browse your Piwigo albums from Canva.
- Add Piwigo photos directly to your design.
- Save Canva exports back to a Piwigo album as JPG or PNG.
- Use your own Piwigo instance with a local connector token.

## Reviewer notes

This app requires the Canva Connector plugin to be installed on a Piwigo instance. Reviewers can use the provided test Piwigo URL and connector token to test the full flow.

Piwigo is self-hosted software with user-controlled domains. To avoid routing user media or Piwigo credentials through a central third-party service, this app uses an open-source connector plugin installed on the user's own Piwigo instance. The connector generates a local revocable token after the Piwigo administrator reviews and accepts the requested access.
