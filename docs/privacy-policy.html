Overview

Piwigo Media for Canva connects Canva to a user-controlled Piwigo instance through the open-source Canva Connector plugin installed on that Piwigo instance.

The app is designed to avoid collecting Piwigo passwords, Piwigo API keys, or central copies of user media.

Data processed by the Canva app

The Canva app stores the Piwigo URL and connector token in the user's browser storage so the user can reconnect without entering them again. The connector token is created and revoked on the user's Piwigo instance.

Data processed by the Piwigo Connector

The connector can list albums, serve selected image files to Canva, and upload Canva exports to a selected Piwigo album. These actions happen between Canva and the user's own Piwigo instance.

Data not collected

The app does not ask for or store Piwigo passwords. The app does not ask for or store Piwigo API keys. User photos are not routed through a central Piwigo Media server.

Revoking access

Users can revoke connector tokens from their Piwigo Connector admin page at any time.

Contact

For privacy questions, open an issue at GitHub Issues.
