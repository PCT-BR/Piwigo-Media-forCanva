# Visual Assets Brief

## Needed assets

- App icon / logo for Canva Developer Portal.
- Featured image for Marketplace listing.
- Optional second featured image if Canva asks for a carousel.

## Featured image guidance

Canva's listing guideline recommends a featured image at `2400 x 1800` pixels, 4:3 aspect ratio, with a clear visual representation of the app's functionality.

Recommended direction for Piwigo Media:

- Show a clean photo-library grid on the left.
- Show one selected image flowing into a Canva-style design canvas.
- Keep text minimal or absent.
- Use bright, uncluttered photography.
- Avoid showing the whole Canva editor or too much UI detail.

## Icon direction

Recommended concept:

- Simple rounded square app icon.
- Piwigo-inspired media/gallery idea without copying protected marks.
- Visual metaphor: photo stack + upload/export arrow.
- Clean enough to read at small sizes.

## Asset staging

Save candidate assets in:

```text
assets/marketplace/
```

Suggested filenames:

```text
assets/marketplace/piwigo-media-icon.png
assets/marketplace/piwigo-media-featured-01.png
assets/marketplace/piwigo-media-featured-02.png
```
