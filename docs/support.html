Get help

For installation help, bug reports, or feature requests, use GitHub Issues:

https://github.com/PCT-BR/Piwigo-Media-forCanva/issues

Before opening an issue

  • Confirm that the connector folder is installed at plugins/canva_connector.
  • Confirm that your Piwigo instance is available over HTTPS.
  • Confirm that you generated a connector token from a Piwigo administrator account.
  • Include your Piwigo version and the error message shown in Canva.
